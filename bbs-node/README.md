# bbs-node

## 开发

**依赖**

- Node.js v6 LTS
- yarn

### 安装依赖

使用 `yarn` 管理模块依赖

```sh
npm i -g yarn
```

安装依赖

```sh
yarn
```

```sh
# 增加依赖(生产依赖)
yarn add koa

# 增加开发依赖
yarn add webpack --dev

# 移除依赖
yarn remove webpack
```

### 本地开发

```sh
yarn start
```
